# castle-war

Youtuber KUN 様の攻城戦を統合版で再現したビヘイビアパックです。

---

## 導入方法（ワールド管理者向け）

1. `攻城戦BP` フォルダを Minecraft の `behavior_packs` フォルダに入れる
2. ワールドの設定からビヘイビアパックを有効化する
3. 実験的なゲームプレイの **ベータAPI** をオンにする
4. Minecraftの実験的機能の **スクリプトAPI** を有効にする

### 初回セットアップ

ワールドに入ったら、以下のコマンドを一度だけ実行してください。


これでスコアボードが作成されます。

## 遊び方

### ゲームの流れ

1. **チーム分け** — `game:set` でプレイヤーを赤・青にランダム振り分け
2. **リーダー投票** — 各チームで投票し、リーダーを決定（30秒）
3. **建築フェーズ** — 各チームの陣地に要塞を建築（600秒）
4. **攻守決め** — ランダムで先攻・後攻を決定
5. **第1ラウンド** — 攻撃チームが敵リーダーを狙う（5分）
6. **攻守交代**
7. **第2ラウンド** — チームを入れ替えて再戦（5分）
8. **結果発表** — 防衛時間が長かったチームの勝利

### 管理者コマンド

| コマンド | 内容 |
|---|---|
| `/scriptevent game:setup` | スコアボードを初期化（初回のみ） |
| `/scriptevent game:set` | プレイヤーをチームに振り分け |
| `/scriptevent game:start` | ゲームを開始 |

### プレイヤーコマンド（チャットに入力）

| コマンド | 内容 |
|---|---|
| `.help` | コマンド一覧を表示 |
| `.status` | 自分のプレイ回数を表示 |
| `.sui` | 自害する |

### 勝利条件

- 相手チームの **リーダーを倒した時点の経過時間** を比較
- 防衛時間が**長かったチームの勝利**
- 制限時間（5分）内にリーダーを倒せなかった場合は時間切れ
